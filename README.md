## 职教圈APP
目标是首先适配安卓，然后适配微信小程序

## 业务目录
personal 个人中心相关  
special 热门专题下面的页面  
indexIcon 首页轮播图下面的icon相关页面  
information 资讯相关页面  
login  登录相关页面  
schoolDetails  学校详情
